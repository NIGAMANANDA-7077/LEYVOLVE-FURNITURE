# Livo Living - Luxury Real Estate Landing Page

A modern, high-performance landing page built with React, Tailwind CSS, and Framer Motion.

## Features
- **Responsive Design**: Mobile-first approach.
- **Visuals**: Smooth animations, glass-morphism, and parallax effects.
- **Architecture**: Clean component-based structure.

## Tech Stack
- React
- Tailwind CSS (v3)
- Framer Motion
- Lucide React

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Start Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Customization
- **Tailwind Config**: Edit `tailwind.config.js` to change valid colors and fonts.
- **Content**: Update component files in `src/components/` to change text and images.
